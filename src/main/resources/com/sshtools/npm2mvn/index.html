Npm2Mvn

A bridge between Npm and Maven

Setup

Repository

First you must configure the npm2mvn repository. Just add the following to your project pom.xml 

       <repository>
           <id>npm2mvn</id>
           <url>${serverUrl}</url>
       </repository>

Add Your Dependencies

Now add your dependencies. You just need the package name and the version. The groupId for npm2mvn dependencies will always be npm, unless the package is in an organisation, in which case it will be npm.orgName

       <dependency>
           <groupId>npm</groupId>
           <artifactId>bootstrap</artifactId>
           <version>5.3.1</version>
       </dependency>
       <dependency>
           <groupId>npm.fortawesome</groupId>
           <artifactId>fontawesome-free</artifactId>
           <version>6.4.0</version>
       </dependency>

Access The Resource

You can now access the Npm packages resources from within Java code.

       try(var in = getClass().getResourceAsStream("/web/framework/npm/bootstrap/5.3.1/css/bootstrap.css")) {
          in.transferTo(response.getOutputStream());
       }
In the real world, of course you wouldn't want to map every single resource like this, you would have your framework such as Spring Web or plain old Servlets map the classpath to some namespace in the web URI.
